Что такое DNS (Domain Name System) и в чём разница протоколов HTTP/HTTPS и их версий - краткое эссе, своими словами.

DNS - Это компьтерная распределенная система для получения информации о доменах. Чаще используется для получения IP адреса по имени устройства, маршрутизации почты.

HTTP/HTTPS - Это протоколы, которые передают запросы от юзер агента (которым выступает браузер) к серверу Или DNS для запроса IP. 
Или, когда браузер собирает сайт для обычного пользователя из разных источников.

HTTPS и HTTP отличаются тем, что буква S - поддерживает шифрование.
